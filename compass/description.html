Computational Modelling of Primary School Segregation (COMPASS)

Interact with the widgets to change the parameter values of the model. Clicking the reset button will update the changed values. The residential process only runs for the 'lattice' case and will run first until convergence or until the maximum amount of steps is reached. After this, schools are chosen, taking the residential location as given. For more information please visit the COMPASS website or contact e.p.n.dignum@uva.nl.
